# SAP Order-to-Cash & SAP BTP

## Business Problem

An organisation required a structured sales order process to manage customer orders from initial order creation through delivery and billing, while improving process visibility and reducing manual handling.

## Solution

An end-to-end SAP S/4HANA Sales and Distribution Order-to-Cash process was simulated alongside an SAP BTP Build Apps order management solution.

## SAP S/4HANA Order-to-Cash Process

- Sales order creation
- Outbound delivery processing
- Goods issue
- Billing and invoice creation
- End-to-end process validation

## SAP BTP Build Apps Solution

- Create Order application
- Customer, product, quantity, and pricing data capture
- Automated total amount calculation
- Order status tracking
- SalesOrders data entity
- Order creation and retrieval logic

## Business Analysis Approach

The project demonstrates process analysis, end-to-end workflow understanding, application configuration, data structure definition, testing, and documented solution evidence.

## Outcome

The solution demonstrates understanding of SAP SD Order-to-Cash processes and low-code application development supporting structured order management and improved process visibility.
