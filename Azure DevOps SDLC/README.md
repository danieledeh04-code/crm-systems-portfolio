# Azure DevOps SDLC Portfolio

## Overview

This project demonstrates practical Agile SDLC delivery using Azure DevOps to structure business requirements, manage backlog items, and provide visibility across the delivery lifecycle.

## Business Problem

A digital booking solution required a structured delivery approach to organise business requirements, break work into manageable delivery items, and support cross-functional delivery tracking.

## Solution

Azure DevOps was used to translate business requirements into epics, features, user stories, acceptance criteria, and delivery tasks.

## Agile Delivery Evidence

- Epic and feature breakdown
- User story development
- Gherkin-style acceptance criteria
- Backlog management
- Task breakdown and delivery tracking
- Sprint board organisation
- Delivery planning and work item visibility
